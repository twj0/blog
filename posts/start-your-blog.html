前言

今天开始搭建自己的个人博客，选择了安知鱼的 Hexo 主题，计划部署到 Cloudflare Pages 上。

技术栈选择

  • 静态站点生成器: Hexo
  • 主题: hexo-theme-anzhiyu
  • 内容管理: GitHub 仓库
  • 部署平台: Cloudflare Pages
  • 域名: 自定义域名

项目结构

blog-theme-anzhiyu/
├── hexo-theme-anzhiyu/     # 安知鱼主题文件
├── blog/                   # 博客文章目录
│   └── 2025/
│       └── 08/
├── posts/                  # 静态文章页面
├── css/                    # 样式文件
└── index.html             # 主页

部署优势

  • 🚀 极速部署 - 直接部署静态文件，无需构建过程
  • ⚡ 快速访问 - Cloudflare 全球 CDN 加速
  • 💰 零成本 - 完全免费的解决方案
  • 🔄 自动更新 - 推送到 GitHub 自动部署
  • 🛡️ 高可用 - Cloudflare 的可靠性保障
