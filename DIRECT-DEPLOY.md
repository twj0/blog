## 🎯 Cloudflare Pages 设置

### 方法一：自动部署（推荐）
1. 在 Cloudflare Pages 中连接你的 GitHub 仓库
2. **构建设置**：
   - 框架预设：`None`
   - 构建命令：留空（不需要）
   - 构建输出目录：`/`（根目录）
   - 根目录：`/`

### 方法二：手动上传
1. 将以下文件上传到 Cloudflare Pages：
   - `index.html`
   - `css/` 目录
   - `posts/` 目录
