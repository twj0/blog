# 开始你的博客之旅

## 前言

今天开始搭建自己的个人博客，选择了安知鱼的 Hexo 主题，计划部署到 Cloudflare Pages 上。

## 技术栈选择

- **静态站点生成器**: Hexo
- **主题**: hexo-theme-anzhiyu
- **内容管理**: GitHub 仓库
- **部署平台**: Cloudflare Pages
- **域名**: 自定义域名

## 项目结构

```
blog-theme-anzhiyu/
├── hexo-theme-anzhiyu/     # 安知鱼主题文件
├── blog/                   # 博客文章目录
│   └── 2025/
│       └── 08/
└── src/                    # Hexo 项目源码（待创建）
```
