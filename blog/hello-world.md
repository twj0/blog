欢迎来到我的博客！这是使用 AnZhiYu 主题创建的第一篇文章。

## 关于这个博客

这个博客使用了以下技术栈：

- **Hexo**: 静态站点生成器
- **AnZhiYu**: 精美的博客主题
- **Cloudflare Pages**: 免费的静态站点托管
- **GitHub**: 代码和内容管理

## 特色功能

- 🎨 精美的卡片式设计
- 📱 完全响应式布局
- ⚡ 极速加载体验
- 🔍 全文搜索功能
- 💬 评论系统支持
- 📊 访问统计分析

## 开始写作

你可以通过以下方式创建新文章：

```bash
# 使用 Hexo 命令
hexo new "文章标题"

# 或直接在 blog/ 目录创建 Markdown 文件
```

文章支持丰富的 Markdown 语法和扩展功能。
